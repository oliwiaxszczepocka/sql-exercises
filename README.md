# sql-exercises

I created made up database called 'online_shop' and generated some random data to improve my SQL skills.

It contains following tables: 
orders – Contains order information, including the date the order was placed and its status.
customers – Stores customer information, such as name, surname, city, and email.
details – Stores order details: which products were purchased, in what quantity, and in which order.
products – Stores product information, including name, category, and price.

Here's a code to create tables and generated data included in insert's:

CREATE TABLE customers (
    id_ NUMERIC PRIMARY KEY,
    name_ VARCHAR(50) NOT NULL,
    surname VARCHAR(50) NOT NULL,
    city VARCHAR(100) NOT NULL,
    e_mail VARCHAR(100) NOT NULL
);


CREATE TABLE products (
    id_ NUMERIC PRIMARY KEY,
    name_ VARCHAR(50) NOT NULL,
    category VARCHAR(50) NOT NULL,
    price NUMERIC(6,2) NOT NULL
);


CREATE TABLE orders (
    id_ NUMERIC PRIMARY KEY,
    id_client NUMERIC,
    date_ DATE NOT NULL,
    status VARCHAR(20) NOT NULL,
    CONSTRAINT id_client_fk FOREIGN KEY (id_client) REFERENCES customers(id_)
);


CREATE TABLE details (
    quantity NUMERIC NOT NULL,
    id_order NUMERIC,
    id_product NUMERIC,
    CONSTRAINT id_order_fk FOREIGN KEY (id_order) REFERENCES orders(id_),
    CONSTRAINT id_product_fk FOREIGN KEY (id_product) REFERENCES products(id_),
    CONSTRAINT id_pk PRIMARY KEY (id_order, id_product)
);
